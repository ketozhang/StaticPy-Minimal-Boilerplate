A minimal boilerplate for StaticPy

## Features:
* No CSS framework
* Basic CSS for a good looking but minimal website
* LaTeX/Mathjax enabled
* Homepage links to `/posts`, `/notes` with a boilerplate example of a blogpost and notepage.
